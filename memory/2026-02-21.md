## Philips Hue + Apple HomeKit Fix Found (07:00 UTC)
**Root cause confirmed:** iOS 26.3 / tvOS 26.3 / HomePod 26.3 updates are breaking Matter connectivity with Hue devices.

**Source:** hueblog.com (Feb 16, 2026) - author confirmed Hue devices became inaccessible after HomePods updated to 26.3.

**WORKAROUND FOUND:** Do NOT add Hue Bridge as Matter accessory. Use native HomeKit instead:
1. Remove Hue Bridge from HomeKit → Home Settings → Home Hubs & Bridges
2. Hue App → Settings → Smart home → Unlink HomeKit
3. HomeKit → Add Accessory → More options
4. Select "Philips Hue HomeKit" (NOT the one under "Other Matter Accessories")
5. Use code on back of Hue Bridge

**Status:** No official fix from Apple or Philips Hue yet. Workaround confirmed working by multiple users.
