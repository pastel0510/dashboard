# push-md-files

Commit and push all `.md` files from the workspace to the md-files git repository.

## What it does

1. Adds all `.md` files from `~/.openclaw/workspace/` to git
2. Commits with a timestamped message
3. Pushes to `git@gitgud.io:unreached2457/md-files.git`

## Usage

Run directly - no parameters needed.
