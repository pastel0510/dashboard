Netis AC1200 Router Exposes Full Configuration Without Authentication

An HTTP GET request to /cgi-bin/skk_get.cgi on the Netis AC1200 Router NC21 V4.0.1.4296 returns the entire router configuration as JSON without any authentication. The /cgi-bin/skk_set.cgi endpoint on the Netis AC1200 Router NC21 V4.0.1.4296 feeds the password and new_pwd_confirm POST parameters directly to the OS shell without sanitization.
