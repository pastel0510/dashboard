CVE-2026-44183 · May 12, 2026

Cleanuparr: X-Forwarded-For leftmost parsing allows remote unauthenticated admin takeover when reverse-proxy mode is enabled

TrustedNetworkAuthenticationHandler in Cleanuparr parses the leftmost X-Forwarded-For header entry as the client IP. An attacker can spoof a local IP value in this header to pass trusted-network checks and automatically log in as the administrator without credentials when reverse-proxy mode is enabled. Fixed in 2.9.10.

Buffer overflow causing application termination on Apple OSes

A buffer overflow due to insufficient bounds checking in Apple iOS, iPadOS, macOS, tvOS, visionOS, and watchOS allows a remote attacker to cause unexpected app termination, resulting in denial-of-service.
