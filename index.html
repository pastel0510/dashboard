CVE-2026-39888 · Apr 8, 2026

PraisonAI Sandbox Escape via Exception Frame Traversal (CRITICAL)

The execute_code() function in praisonaiagents defaults to sandbox_mode="sandbox" which runs user code in a subprocess. The AST blocklist embedded in the subprocess wrapper contains only 11 attribute names, missing four attributes (__traceback__, tb_frame, f_back, f_builtins) that form a frame-traversal chain. Chaining these through a caught exception exposes the real Python builtins dict, allowing exec retrieval and RCE. Fixed in version 1.5.115.

PraisonAI Remote Code Execution via YAML Deserialization (CRITICAL)

The AgentService.loadAgentFromFile method uses the js-yaml library to parse YAML files without disabling dangerous tags (!!js/function, !!js/undefined). Attackers can craft malicious YAML files that execute arbitrary JavaScript code via the API endpoint, leading to RCE. Affects versions prior to 4.5.115.

CVE-2026-40035 · Apr 8, 2026

Unfurl Flask Debug Mode Activation (CRITICAL)

Unfurl through 2025.08 contains improper input validation in config parsing that enables Flask debug mode by default. The debug config value is read as a string and passed directly to app.run(), causing any non-empty string to evaluate truthy. Attackers can access the Werkzeug debugger to disclose sensitive info or achieve RCE.
