Claude Code: Sandbox Escape via Symlink Following Allows Arbitrary File Write Outside Workspace

Claude Code's sandbox did not prevent sandboxed processes from creating symlinks pointing outside the workspace. CVE-2026-40496 · April 21, 2026 (updated April 23) · CVSS 9.1

FreeScout Predictable Attachment Token Allows Unauthenticated Private File Download

FreeScout generates attachment download tokens using a weak formula (md5(APP_KEY + attachment_id + size)). Since attachment IDs are sequential and size is brute-forceable, unauthenticated attackers can forge tokens and download any private attachment without credentials. Fixed in version 1.8.213.

CVE-2026-40575 · April 22, 2026 · CVSS 9.1

OAuth2 Proxy Authentication Bypass via X-Forwarded-Uri Header Spoofing

OAuth2 Proxy versions 7.5.0 through 7.15.1 may trust a client-supplied X-Forwarded-Uri header when reverse-proxy mode and skip-auth rules are configured. Attackers can spoof this header to make the proxy evaluate authentication against a different path, bypassing access controls and accessing protected routes without a valid session.
