CVE-2026-44825 · June 1, 2026 · GCVE · OpenCVE

Apache Solr Hardcoded Credentials in Basic Authentication Setup Tool

Apache Solr versions 9.4.0 through 9.10.1 and 10.0.0 have hardcoded credentials in the bin/solr auth enable Basic Authentication setup tool. The tool silently installs publicly known default credentials (superadmin, admin, search, index) alongside the user-specified account, allowing a remote attacker to gain full administrative access to the Solr cluster. Immediate workaround: delete the template users from security.json. Fixed in upcoming versions 9.11.0 and 10.1.0.

TOTOLINK N300RH Stack-Based Buffer Overflow in setWiFiBasicConfig

TOTOLINK N300RH version 6.1c.1353_B20190305 contains a stack-based buffer overflow in the setWiFiBasicConfig function within the wireless.so web management interface component. Manipulation of the KeyStr argument allows unauthenticated remote attackers to trigger the overflow over the network. Public PoC exploit has been released. CVSS 3.1: 9.8 (CRITICAL).

CVE-2018-25427 · June 1, 2026 · GCVE · OpenCVE

Arm Whois 3.11 Stack-Based Buffer Overflow via SEH Overwrite

Arm Whois 3.11 contains a stack-based buffer overflow that allows remote attackers to execute arbitrary code by supplying oversized input (exceeding 658 bytes with shellcode) to the IP address or domain field, overwriting the structured exception handler to gain command execution. Public exploit code is available on ExploitDB and VulnCheck has published an advisory. CVSS 4.0: 9.3 (CRITICAL).
