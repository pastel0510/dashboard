CVE-2026-6951 · April 25, 2026 · [KEV] No

simple-git Remote Code Execution via Incomplete Fix

Versions of the package simple-git before 3.36.0 are vulnerable to Remote Code Execution (RCE) due to an incomplete fix for CVE-2022-25912 that blocks the -c option but not the equivalent --config form. If untrusted input can reach the options argument passed to simple-git, an attacker may still achieve remote code execution by enabling protocol.ext.allow=always and using an ext:: clone source.

CVE-2026-41044 · April 24, 2026 · [KEV] No

Apache ActiveMQ Code Injection via Malicious Broker Name

An authenticated attacker can use the admin web console page to construct a malicious broker name that bypasses name validation to include an xbean binding. This can be later used by a VM transport to load a remote Spring XML application, leading to arbitrary code execution on the broker's JVM through bean factory methods such as Runtime.exec(). Affects Apache ActiveMQ before 5.19.6 and 6.0.0 before 6.2.5.

CVE-2026-40466 · April 24, 2026 · [KEV] No

Apache ActiveMQ Code Injection Bypass via HTTP Discovery Transport

An authenticated attacker may bypass the fix in CVE-2026-34197 by adding a connector using an HTTP Discovery transport via BrokerView.addNetworkConnector or BrokerView.addConnector through Jolokia if the activemq-http module is on the classpath. A malicious HTTP endpoint can return a VM transport through the HTTP URI which bypasses validation, leading to arbitrary code execution on the broker's JVM. Affects Apache ActiveMQ before 5.19.6 and 6.0.0 before 6.2.5.
