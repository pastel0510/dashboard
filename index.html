CVE-2026-40492 · Apr 18, 2026 · updated Apr 20, 2026

SAIL Library XWD Codec Buffer Overflow (CRITICAL 9.8)

SAIL is a cross-platform library for loading and saving images. The XWD codec resolves pixel format based on pixmap_depth but the byte-swap code uses bits_per_pixel independently. When pixmap_depth=8 but bits_per_pixel=32, the byte-swap loop accesses memory as uint32_t*, reading/writing 4x the allocated buffer size, causing heap overflow.

CVE-2026-40493 · Apr 18, 2026 · updated Apr 20, 2026

SAIL Library PSD Codec Buffer Overflow (CRITICAL 9.8)

SAIL library PSD codec computes bytes-per-pixel (bpp) from raw header fields channels * depth, but the pixel buffer is allocated based on the resolved pixel format. For LAB mode with channels=3, depth=16, bpp is calculated as 6 but the actual buffer is smaller, leading to out-of-bounds read/write.

SAIL Library TGA Codec RLE Decoder Overflow (CRITICAL 9.8)

SAIL library TGA codec's RLE decoder in tga.c has an asymmetric bounds check vulnerability. The run-packet path correctly clamps the repeat count to remaining buffer, but the raw-packet path lacks proper bounds checking, allowing out-of-bounds memory access when processing crafted TGA files.
